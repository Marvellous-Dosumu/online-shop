# Online Shop

This project is a simple static online shop built with HTML, CSS, and JavaScript.

## Publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload this project to the repository.
3. In the repository settings, open Pages.
4. Choose the branch you pushed (usually `main`) and the root folder `/`.
5. Save and wait for GitHub to publish the site.

Your site will be available at:
https://<your-username>.github.io/<your-repository-name>/
